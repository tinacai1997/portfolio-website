Design Portfolio

Creating a home for my project work

Overview

How can I share my UX work, skills, and experiences with others?

My design portfolio is an online platform that displays my work, design skills, and personality to other designers, potential recruiters, and hiring managers.
It's also a medium for me to practice and demonstrate front-end development and web design.
Role: Front-End Dev, Web/UX/UI Designer
Tools: Adobe XD, VS Code, Atom, GitHub, Google Sheets
Timeframe: Nov 2021 - Present

HTML

Creating the initial structure for my portfolio website

I began my website by writing HTML documentation using the IDE, Atom. To ensure accessibility, I used a mobile-first approach. To focus on content over style, I referenced sample wireframes.

Sample Wireframes Structure

GITHUB

Tracking development changes with a GitHub repository

I used GitHub Desktop to create a repository for my website. Though this was a solo project, I wanted to learn how to use GitHub to better work with developers and practice keeping a repo updated.
Visit Project GitHub

DESIGN

Designing wireframes to guide my website design

With the structure finished, I began writing CSS to emulate the sample wireframes. I noticed I was constantly tweaking the style as I was writing. Therefore, I decided to create wireframes to guide my design.
To ensure my site was usable across devices, I created mobile and x-large wireframes then adjusted the CSS accordingly. Seeing my design from the development perspective revealed the importance of creating clear and consistent assets as a designer.

TESTING + ITERATION

Most test participants thought my site design was too plain

I observed whether five usability test participants could figure out how to learn about my projects, me, my work history, and my contact info using my site. Participants had no issue with the test scenarios, but all commented on my site being too plain.
To address this, I added dynamic page colors and parallax images to add dynamics to my portfolio. These additions were well received when critiquing my site with other designers.
